# SwissWayExplorer Pilot Brief

## Goal
Launch `swisswayexplorer` as the pilot of a reusable Hermes-backed site factory.

## Scope (Phase 1)
- Create reusable Astro scaffold (layout + sections + config)
- Establish project documentation and operational memory files
- Replace starter homepage with seed content sections

## Non-Goals (Phase 1)
- No Sanity integration yet
- No dependency changes
- No production cutover actions

## Success Criteria
- Homepage is no longer Astro starter
- Reusable structure exists for future sites
- Docs and memory standards are present in repo
